## 進化の仕組み

ペットはお世話の品質によって異なるキャラクターに進化します。
進化を決める2つの要素:

### 1. お世話ミス

進化を左右する最重要の隠しパラメータ。以下の場合にカウントされます:

| 条件 | 猶予時間 |
|------|---------|
| おなかが0のまま放置 | 約15分 |
| ごきげんが0のまま放置 | 約15分 |
| 病気を放置 | 即時〜数時間 |
| うんち長時間放置 | 10〜15分 |
| 寝たのに消灯しない | 約15分 |
| わがまま呼び出しを無視 | 約15分 |

### 2. しつけメーター

正しいタイミングでしつけると +25%。最大100%。

</div>

<div class="section">

## 進化ティア

お世話ミスの回数としつけメーターで進化先のティアが決まります。

| ティア | 条件 | 結果 |
|--------|------|------|
| ⭐ Tier 1 | ミス 0〜1回 + しつけ100% | 最良キャラ |
| ⭐ Tier 2 | ミス 1〜2回 + しつけ75% | 良キャラ |
| Tier 3 | ミス 3〜5回 + しつけ50% | 普通キャラ |
| Tier 4 | ミス 6回以上 | 残念キャラ |

## 育成ティップス

### Tier 1 を目指すには

1. **おなか・ごきげんをこまめにチェック** — 0になる前にお世話する
2. **うんちは即そうじ** — 放置は病気の原因
3. **しつけのタイミングを見極める** — わがままの時だけしつける
4. **寝たらすぐ消灯** — 15分以内に対応
5. **おやつは控えめに** — 太りすぎは進化に悪影響

### 死亡を避けるには

| 死因 | 予防 |
|------|------|
| 餓死 | おなか0を数時間放置しない |
| 病死 | 病気は12時間以内に治療 |
| 蓄積ダメージ | お世話ミスを減らし寿命を延ばす |
| 老衰 | 完璧なお世話でも避けられない（寿命7〜25日） |

</div>

<div class="section">

## 寿命

お世話品質により **約7〜25日間**。

- 完璧なお世話: 最長25日
- 普通のお世話: 約14日
- 放置気味: 約7日

世代を重ねるとボーナスが得られます。ペットが亡くなっても、次のたまごで新しい冒険が始まります。
