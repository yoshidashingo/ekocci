## ライフサイクル

ペットはたまごから生まれ、お世話次第で様々な姿に成長します。

| 段階 | 期間 | 説明 |
|------|------|------|
| 🥚 たまご | 約5分 | 画面上でたまごが揺れ、自動的に孵化 |
| 👶 あかちゃん | 約1時間 | 常にお世話が必要。この段階では死なない |
| 🧒 こども | 約24時間 | ミニゲーム解禁。頻繁にお世話を求める |
| 🧑‍🎤 ヤング | 約2〜3日 | 進化分岐。お世話品質で次の姿が変わる |
| 🧑 おとな | 寿命まで | 最終形態。キャラクターが確定 |
| 👴 シニア | おとな後期 | 長寿ペットのみ到達 |

## Apple Watch ならでは

- **コンプリケーション** — 文字盤からペットの状態をチェック
- **Taptic Engine** — ごはん、進化、ミニゲームをハプティクスで体感
- **Digital Crown** — メニュースクロール、ミニゲーム操作
- **常時表示** — Always-On Display でペットがいつもそばに
